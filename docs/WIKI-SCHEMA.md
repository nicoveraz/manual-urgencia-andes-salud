## Notas técnicas

### Creación de páginas

Las páginas fueron creadas con el script `scripts/create_wiki_pages.py` vía GraphQL API de Wiki.js.
Resultado: **70 creadas · 1 ya existía (home) · 0 fallidas** — ejecutado 2026-03-08.

```bash
# Recrear/actualizar páginas desde cero
scp /tmp/create_wiki_pages.py ubuntu@146.235.242.103:/tmp/
ssh -i ~/Downloads/ssh-key-2026-01-27.key ubuntu@146.235.242.103
# (generar /tmp/wiki_token.txt con JWT válido — ver ESTADO-ACTUAL.md)
python3 /tmp/create_wiki_pages.py
```

### Convención de edición

- Editor: Markdown (no Visual Editor)
- Locale: `es` en todas las páginas
- Publicadas (`isPublished: true`), no privadas
- Tags: vacíos en la creación inicial — agregar según contenido

### Agregar nuevas páginas

1. Añadir entrada al array `PAGES` en `scripts/create_wiki_pages.py`
2. Actualizar este archivo (`docs/WIKI-SCHEMA.md`) con la nueva entrada
3. Ejecutar el script en el servidor (solo crea páginas nuevas; las existentes son skip)

### Eliminar o mover páginas

Usar la interfaz de administración de Wiki.js (`caspm.urgpedia.cl/a/`) o via GraphQL:
```graphql
mutation { pages { delete(id: PAGE_ID) { responseResult { succeeded } } } }
```
